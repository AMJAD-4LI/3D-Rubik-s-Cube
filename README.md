# 🧩 3D Rubik's Cube

An interactive **3D Rubik's Cube** built with **HTML, CSS, and JavaScript**. The project features realistic 3D transformations, smooth animations, and responsive controls, delivering an engaging browser-based cube experience with clean and optimized code.

## ✨ Features

- 🧩 Interactive 3D Rubik's Cube
- 🎨 Realistic 3D Transform Effects
- 🔄 Smooth Cube Rotations & Animations
- 🖱️ User-Friendly Controls
- 📱 Fully Responsive Design
- ⚡ Lightweight & Optimized Performance
- 🌐 Cross-Browser Compatibility

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```text
3D-Rubik-s-Cube/
│── index.html

│── script.js
└── README.md
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/AMJAD-4LI/3D-Rubik-s-Cube.git
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.

## 📸 Preview

Experience a fully interactive 3D Rubik's Cube with smooth animations and responsive controls directly in your browser.
